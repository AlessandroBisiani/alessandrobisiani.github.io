Reinforcement Learning Simulator - June 2015

My thesis project at City University London. Single agent reinforcement learning simulator in Java. Made to be applicable over a wide range of experiment sizes and complexities. Allows the arbitrary definition of an abstract environment model and a reward function. Includes Q-learning and SARSA algorithms runnable with ε-greedy or softmax policies. A pedagogical and practical tool for students and researchers that would allow observation of the process of learning, and allow convenient exportation of learning data and experiment state saving. Still missing: Multi-agent learning, multiple initial and goal states, GUI cleanup (especially for the save funtion), sensible refactor to support simple algorithm plugins.
